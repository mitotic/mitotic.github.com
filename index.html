  • code.mindmeldr Blog: A blog about open source components of MindMeldr

  • GraphTerm: A Graphical Terminal Interface that extends xterm by adding GUI-like features and session sharing

  • OTrace: An object-oriented python debugger for nonlinear tracing

These are open source components developed as part of MindMeldr, a live group interaction framework that allows you to present slides, photos, or questions to a group of users over the internet using a browser. Members of the group can respond instantly using their own browser, or by "tweeting" from a mobile phone. The responses can be in the form of text messages, graphical annotations (i.e., marking a point or drawing on the slide), or answers to multiple choice questions.
